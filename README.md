# Drum Machine

A simple and interactive Drum Machine web app built with HTML, CSS, and JavaScript.

## Project Overview

This project is a functional drum machine that allows users to play drum sounds either by clicking on pads or pressing specific keys on the keyboard. It mimics classic drum machine behavior with a clean, responsive design and real-time audio playback.

## Features

- 9 drum pads arranged in a 3x3 grid
- Each pad corresponds to a keyboard key: Q, W, E, A, S, D, Z, X, C
- Click or press the associated key to play a drum sound
- Visual feedback when a pad is triggered
- Displays the name of the sound currently playing
- Responsive and accessible design

## How to Use

1. Open `index.html` in any modern web browser.
2. Click on any drum pad or press the corresponding key on your keyboard.
3. Listen to the drum sounds and watch the display update with the sound name.
4. Enjoy making beats!

## Development

- Built with vanilla JavaScript, HTML5, and CSS3.
- Audio clips sourced from [freeCodeCamp Drum Machine Curriculum](https://cdn.freecodecamp.org/curriculum/drum/).
